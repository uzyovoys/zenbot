What is Zenbot

Zenbot is all-in-one solution for building a conversational chatbots or natural language interface for your app or website.

Instead of making everything on your own, you can use Zenbot - it provides an NLP tools and bot hosting in a single service. Moreover, it is already connected with Facebook Messenger, Slack, Telegram and other messengers! Thus you have not to implement all of these messengers APIs.

Zenbot features

Who can develop with Zenbot

Zenbot provides a chatbot framework for developers who are familiar with programming a bit, but didn't dive into NLP or AI yet.

With Zenbot you can use your own Javascript code, thus it is better if you have some experience with it. But it is not mandatory - you can still create a great bots without programming.
